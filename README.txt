This is a android application that uses user geo location to find people that are nearby and uses Facebook OAuth to authenticate user and uses GoogleAPI to map location where 2 users met. 

GoblinLocation Folder contains Server Code
MyApplicationTestGPS contain Code for Android Application
simple.sql contains Database 

To run MyApplicationTestGPS in android the GlobalStrings Class contains the url variable that contains the url to host which runs the server so it needs to changed to the server running host ip-adress. 

Make sure in the GoblinLocation server code, put Your Facebook APP Secret and ID in oauth/Connect.java
